About

Angular By Example is a site that showcases issues and solutions found during Angular development.

How do I use it?

Go to the examples section and find examples of issues you want to know more about. See the specific examples for info on ways to tackle that specific issue. You will see examples on screen. Also there is a repo of this site available on github so that you can see the code used to generate this site, which provides a behind the scenes look at the examples in action.

Contributing

Go to the issues section of the repo and see ways that you can contribute to this project.

Also, you can contribute by adding issues for examples you would like to see added to the site.
